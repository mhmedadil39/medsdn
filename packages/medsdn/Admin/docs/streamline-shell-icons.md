# Streamline Shell Icons

This document tracks the Streamline Plump Color SVG assets selected for the MedSDN Admin shell migration.

Scope in this phase:

- header visit-shop shortcut
- desktop sidebar primary destinations
- mobile sidebar primary destinations
- quick creation shortcut cards

These SVG files are stored in `src/Resources/assets/images/streamline-plump-color/` and are consumed by the shared shell icon Blade component.

The wider Admin application still uses `icomoon` for operational glyphs outside the shell.
