# Reporting Security Issues

MedSDN Headless eCommerce values the contributions of the security research community, and we look forward to working with you to minimize risk to MedSDN merchants.

## Where should I report security issues?

If you think that you have found a security issue in MedSDN Headless eCommerce, please do not use the issue tracker and do not post it publicly. Instead, all security issues must be sent to mailto:security@medsdn.com.
